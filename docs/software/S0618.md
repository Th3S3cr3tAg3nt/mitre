# FIVEHANDS
## ID:S0618
[FIVEHANDS](/mitre/software/S0618) is a customized version of [DEATHRANSOM](/mitre/software/S0616) ransomware written in C++. [FIVEHANDS](/mitre/software/S0618) has been used since at least 2021, including in Ransomware-as-a-Service (RaaS) campaigns, sometimes along with [SombRAT](/mitre/software/S0615).(Citation: FireEye FiveHands April 2021)(Citation: NCC Group Fivehands June 2021)
